# LinuxScripts 🐧

A personal collection of useful scripts and daemons for Linux systems.

## Details

This repo provides a several linux scripts and daemon you can use, speccially if you are using/configuring a WM like Hyprland, BSPWM, i3, etc.

For example:
  - check_battery_level is a daemon for checking and notify when battery levels reaches a low or critical level.
  - raise_volume is a script to increase system volume. So you can configure a keymap in your WM config file to run it.

## Requirements

Some scripts use Nerd Icons, so I suggest to install any [Nerd Font](https://www.nerdfonts.com/) or edit the scripts directly.
